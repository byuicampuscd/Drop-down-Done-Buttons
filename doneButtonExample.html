Done Buttons

Insert the following HTML in-between the drop-down div and the drop-content div:

<div class="check-wrapper">
  <input id="check1" class="check" type="checkbox">
  <label for="check1">DONE</label>
</div>

NOTE: It is important that for each dropdown you add you increment the number in the input id and label for attribute. So the next dropdown will have an input with id="check2" and a label tag with for="check2".
